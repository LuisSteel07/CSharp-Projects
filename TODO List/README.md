# TODO List

Este mini-proyecto trata sobre la creación de un mini administrador de Tareas.

## Módulos empleados
En este proyecto se hace uso de el modulo **System.Data.SQLite** el cual permite la conexión con la DB de Sqlite.

|description|Text   |
